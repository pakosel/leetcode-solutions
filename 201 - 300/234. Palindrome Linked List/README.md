# 234. Palindrome Linked List

Given the `head` of a singly linked list, return `true` if it is a palindrome.

**Constraints**:
- The number of nodes in the list is in the range <code>[1, 10<sup>5</sup>]</code>.
- `0 <= Node.val <= 9`

### Example 1:
```
Input: head = [1,2,2,1]
Output: true
```

### Example 2:
```
Input: head = [1,2]
Output: false
```

### Example 3:
```
Input: root = [2,1], p = 2, q = 1
Output: 2
```
