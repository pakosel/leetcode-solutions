# 5. Longest Palindromic Substring

Given a string `s`, find the *longest palindromic substring* in `s`.

**Constraints**:
- `1 <= s.length <= 1000`
- `s` consist of only digits and English letters.

### Example 1:
```
Input: "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```

### Example 2:
```
Input: "cbbd"
Output: "bb"
```
