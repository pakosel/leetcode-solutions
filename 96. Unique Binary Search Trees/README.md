# 96. Unique Binary Search Trees

Given an integer `n`, return *the number of structurally unique ***BST***'s (binary search trees) which has exactly* `n` *nodes of unique values from* `1` *to* `n`.

**Constraints**:
- `1 <= n <= 19`

### Example 1:
```
   1     1       1      1    3
    \     \     / \    /    /
     3     2   2   3  2    1
    /       \        /      \
   2         3      3        2

Input: n = 3
Output: 5
```

### Example 2:
```
Input: n = 1
Output: 1
```
